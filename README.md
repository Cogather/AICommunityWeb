# AI社区平台

AI社区平台是一个集成了前端和后端的完整项目，提供AI工具分享、实践交流、荣誉殿堂、活动管理等功能的社区平台。

## 项目结构

```
AICommunityWeb/
├── frontend/                # 前端项目 (Vue 3 + TypeScript + Vite)
│   ├── src/
│   │   ├── components/      # 公共组件
│   │   ├── views/           # 页面视图
│   │   ├── mock/            # Mock数据和API模拟
│   │   ├── router/          # 路由配置
│   │   └── utils/           # 工具函数
│   └── ...
├── docs/                    # 项目文档
│   └── api/                 # API接口文档
│       ├── 首页接口文档.md
│       ├── AI使用达人接口文档.md
│       ├── AI优秀实践接口文档.md
│       ├── AI工具专区接口文档.md
│       ├── 扶摇Agent应用接口文档.md
│       ├── 赋能交流接口文档.md
│       ├── 帖子详情与互动接口文档.md
│       ├── 用户认证与信息接口文档.md
│       ├── 消息中心接口文档.md
│       ├── 个人中心接口文档.md
│       └── 管理后台接口文档.md
└── README.md                # 项目说明文档
```

## 技术栈

### 前端
- Vue 3 (Composition API)
- TypeScript
- Vite
- Element Plus
- Vue Router
- Pinia (状态管理)

## 快速开始

### 前端开发

```bash
cd frontend
npm install
npm run dev
```

前端开发服务器默认运行在 `http://localhost:5173`

### Mock模式

项目内置了完整的Mock数据，无需后端即可进行前端开发和测试。Mock数据位于 `frontend/src/mock/index.ts`。

## 功能模块

### 🏠 首页
- Hero轮播图展示
- 荣誉殿堂与AI使用达人（最新获奖者展示）
- 赋能交流板块
- AI优秀实践列表
- 悬浮工具平台
- AI工具专区入口

### 🏆 AI使用达人（荣誉殿堂）
- 个人荣誉墙/时光轴视图
- 团队奖项展示
- 荣誉影响力排行榜
- 送花互动功能
- 奖项规则说明

### 📝 AI优秀实践
- 精华帖子推荐
- 热门帖子排行
- 标签分类筛选
- 部门排名统计
- 热门贡献者展示

### 🔧 AI工具专区
- 多工具切换
- 工具专属帖子列表
- 工具活动展示
- 精华帖子管理

### 🤖 扶摇Agent应用
- 置顶帖子展示
- Agent相关帖子列表
- 最新活动展示

### 💬 赋能交流
- 帖子讨论区
- 标签筛选
- 热门话题

### 📄 帖子详情与互动
- 帖子内容展示
- 点赞/收藏功能
- 评论与回复
- 草稿自动保存

### 👤 用户中心
- 个人资料管理
- 我的帖子/收藏/评论
- 消息通知

### ⚙️ 管理后台
- 轮播图管理
- 荣誉殿堂Banner配置
- AI工具管理

- 个人/团队奖项设置
