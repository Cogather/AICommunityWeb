# 首页接口文档

## 概述

首页需要加载以下数据模块：
1. 首页轮播图（Hero Carousel）
2. 荣誉殿堂（AI使用达人）
3. 赋能交流板块
4. AI优秀实践列表
5. 悬浮工具平台（右侧悬浮面板）
6. AI工具专区列表
7. 工具专区轮播图

---

## 1. 获取首页轮播图

### 接口信息

| 项目 | 说明 |
|------|------|
| 接口路径 | `GET /api/home/carousel` |
| 接口描述 | 获取首页顶部轮播图配置 |
| 权限要求 | 无需登录 |

### 请求参数

无

### 响应数据

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "image": "https://example.com/banner1.jpg",
        "title": "AI 优秀实践",
        "desc": "探索大模型在企业级应用中的最佳落地场景",
        "link": "/practices",
        "showContent": true,
        "order": 1
      }
    ]
  }
}
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 轮播图ID |
| image | string | 是 | 图片URL |
| title | string | 否 | 标题（showContent为true时显示） |
| desc | string | 否 | 描述（showContent为true时显示） |
| link | string | 否 | 点击跳转链接 |
| showContent | boolean | 是 | 是否显示标题和描述内容 |
| order | number | 否 | 排序序号，越小越靠前 |

---

## 2. 获取荣誉殿堂数据

### 接口信息

| 项目 | 说明 |
|------|------|
| 接口路径 | `GET /api/home/honor` |
| 接口描述 | 获取荣誉殿堂配置，包括banner图、奖项列表、Top用户 |
| 权限要求 | 无需登录 |

### 请求参数

无

### 响应数据

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "honor": {
      "bannerImage": "https://example.com/honor-banner.jpg",
      "awards": [
        {
          "id": 1,
          "name": "年度最佳贡献奖",
          "desc": "2026年度",
          "image": "https://example.com/award1.jpg"
        },
        {
          "id": 2,
          "name": "AI创新突破奖",
          "desc": "2026年度",
          "image": "https://example.com/award2.jpg"
        }
      ],
      "topUsers": [
        {
          "id": 1,
          "name": "张三",
          "avatar": "https://example.com/avatar1.jpg",
          "department": "技术部",
          "level": 10,
          "honorCount": 5
        }
      ]
    }
  }
}
```

### 字段说明

**honor 对象：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| bannerImage | string | 是 | 荣誉殿堂Banner图片URL |
| awards | array | 是 | 奖项列表 |
| topUsers | array | 否 | Top用户列表（首页展示前6名） |

**awards 数组元素：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 奖项ID |
| name | string | 是 | 奖项名称 |
| desc | string | 否 | 奖项描述（如年度信息） |
| image | string | 否 | 奖项图片URL |

**topUsers 数组元素：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 用户ID |
| name | string | 是 | 用户姓名 |
| avatar | string | 否 | 用户头像URL |
| department | string | 否 | 所属部门 |
| level | number | 否 | 用户等级 |
| honorCount | number | 否 | 获得荣誉数量 |

## 通用响应格式

### 成功响应

```json
{
  "code": 200,
  "message": "success",
  "data": { ... }
}
```

### 错误响应

```json
{
  "code": 500,
  "message": "服务器内部错误",
  "data": null
}
```

### 常见错误码

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权（需要登录） |
| 403 | 禁止访问（无权限） |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
